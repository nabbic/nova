What does the product assess?

Select the categories that are core to v1. You can pick multiple.

1

Code quality & tech debt

Complexity, duplication, test coverage, linting, dead code. Integrations: SonarQube, GitHub, GitLab, Bitbucket.

2

Security vulnerabilities

SAST, dependency CVEs, secrets in code, OWASP findings, exposed endpoints. Integrations: Snyk, Semgrep, Trivy.

3

Infrastructure & architecture

Cloud footprint, cost, scalability, redundancy, IaC coverage, single points of failure. Integrations: AWS/GCP/Azure APIs, Terraform state.

4

Engineering process maturity

DORA metrics, deployment frequency, incident history, PR review patterns, CI/CD pipeline health. Integrations: GitHub Actions, Jira, PagerDuty.

5

Dependencies & licensing

Open-source license risk, EOL packages, vendor lock-in, third-party API dependencies. Integrations: npm, pip, Maven, SBOM generation.

6

Data & compliance posture

GDPR, SOC 2, HIPAA indicators, PII handling, audit log presence, data residency. Integrations: policy-as-code, cloud config scanners.

7

Documentation & knowledge risk

README quality, architecture docs, API docs, onboarding guides. Bus factor indicators — how much lives in one person's head.
